Proyecto Club-Match

En colaboración con un equipo de tres personas, creamos una aplicación web de página única, altamente receptiva. Esta aplicación permitía a los usuarios registrarse, iniciar sesión y participar en una red social, donde podían crear, editar y borrar comentarios. Las tecnologías utilizadas en este proyecto fueron JavaScript, HTML, CSS y Firebase.
